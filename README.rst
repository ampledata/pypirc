pypirc Management Client


Installation
============

Install from 'the' pypi::

    pip install pypirc


Usage Example
=============

Display current pypi configuration::

    pypirc

Add a pypi server::

    pypirc -s mypypi -u foo -p bar -r http://mypypi.example.com/
